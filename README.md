
Automation Suite

This project is a personal attempt to simplify everyday repetitive tasks using Python-based automation scripts.
It includes different modules for file organization, email automation, web scraping, and system monitoring — all built with the goal of improving productivity and learning real-world automation.


--Features

File management and organization

Email automation using SMTP

Basic web scraping for data collection

Simple system monitoring (CPU, RAM usage etc.)

Easy to extend and customize



--Tech Stack

Language: Python

Libraries Used: os, shutil, smtplib, requests, psutil, bs4

Editor: VS Code



--How to Run

1. Clone the repository

git clone https://github.com/Nitintiwari20/Automation--Suite.git


2. Open the folder in VS Code


3. Install dependencies (if any)

pip install -r requirements.txt


4. Run the main script

python main.py
